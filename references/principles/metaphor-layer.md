# Metaphor Layer

> The metaphor layer is an optional expression and comprehension adapter for `auto-dev-team`. It helps non-technical users understand the current state and lets them keep asking with the same metaphor, but it must never replace the technical source of truth.

## Goal

- translate technical state into a metaphor the user can understand
- allow the user to continue the conversation with the same metaphor
- keep the skill itself anchored on real code, real logs, and real test results
