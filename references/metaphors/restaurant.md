# Standard Metaphor Template: Restaurant / Cooking

## Good Fits

- PM or business-facing communication
- explaining request handling and delivery
- explaining “the system finished internally, but the user still cannot see the result”

## Core Mapping

| Technical object | Restaurant metaphor |
|------------------|---------------------|
| User request | placing an order |
| Brainstorm | confirming the meal and restrictions |
| Architect | arranging the menu and serving order |
| Step | cooking one dish at a time |
| Backend | the kitchen |
| Frontend | serving food to the table |
