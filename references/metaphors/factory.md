# Standard Metaphor Template: Factory / Assembly Line

## Good Fits

- explaining staged processing
- describing status flow and handoff failures

## Core Mapping

| Technical object | Factory metaphor |
|------------------|------------------|
| User request | production order |
| Brainstorm | confirming requirements and tolerances |
| Architect | arranging the production line |
| Step | one workstation at a time |
| Backend | production line |
| Frontend | packaged output shown to the user |
