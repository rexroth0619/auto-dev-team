# Standard Metaphor Template: Logistics / Warehousing

## Good Fits

- explaining handoffs across multiple systems
- describing async delivery and queue behavior
- explaining “processed but not yet delivered”

## Core Mapping

| Technical object | Logistics metaphor |
|------------------|--------------------|
| User request | package order |
| Brainstorm | shipping requirements |
| Architect | route planning |
| Step | one station at a time |
| Backend | sorting center |
| Frontend | final delivery |
