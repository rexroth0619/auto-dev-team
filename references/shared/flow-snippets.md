# Flow Snippets

> Reuse these short blocks to avoid repeating boilerplate in mode docs.

## Critique Summary

```text
🏥 Critique Summary
- Original plan: ...
- Revised plan: ...
- Needs clarification: [none / list]

📌 Next:
[1] Run the original plan
[2] Run the revised plan
[0] Cancel
```

## Phase Confirm (`phase-confirm`)

```text
📍 Current: [phase conclusion]
📌 Next:
[1] Confirm - enter [next phase]
[2] Add details - [missing items]
[0] Cancel
```

## Plan Choice (`plan-choice`)

```text
📍 Current: [plan / critique result]
📌 Next:
[1] Run original plan
[2] Run revised plan
[3] Trust mode
[0] Cancel
```

## Completion Actions (`completion-actions`)

```text
📍 Current: [completed / checkpoint reached]
📌 Next:
[1] Merge / push
[2] Continue development
[3] Continue tests or cleanup
[0] End
```
