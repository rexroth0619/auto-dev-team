# Flow Snippets

> 复用这些短模板，减少模式文档里的重复样板。保留语义，别每个模式都重新写长版。

## 会诊短模板

```text
🏥 会诊结论
- 原计划: ...
- 修订计划: ...
- 待澄清: [无 / 列表]

📌 下一步:
[1] 执行原计划
[2] 执行修订计划
[0] 取消
```

## 快照闸门短模板

```text
💿 执行前快照闸门
- 必须输出 "💿 已保护" 或 "💿 闸门通过" 后才能继续写文件
- 规则以 references/principles/checkpoint-mechanism.md 为准
```

## 测试回执短模板

```text
🧾 测试回执
测试等级: [小测试 / 大测试]
后台自动测试: [方式 / 命令 / 结果 / 证据]
观测对比验证: [档位 / 预期 / 实际 / 差异结论]
前端链路测试: [未触发 / 建议执行 / 已执行 / 暂不可执行]
剩余风险: [无 / 列表]
```

## 下一步短模板

```text
📌 下一步:
[1] 继续当前流程
[2] 切换到更合适的模式
[0] 结束 / 取消
```
